# LusoLaw Marketplace

Uma aplicação web Java (Spring Boot) para marketplace de serviços jurídicos de imigração.

## Pré-requisitos

- Java 17 ou superior
- Maven

## Como executar

1. Clona o repositório
2. Executa `mvn clean compile spring-boot:run`

## Funcionalidades

- Replica a UI do HTML fornecido
- Fluxos de onboarding para clientes e advogados
- Simulação de seleção de serviços e pagamentos

## Implementação atual

- Página HTML estática servida pelo Spring Boot
- Navegação, hero, funcionalidades, como funciona, testemunhos, rodapé
- Modais interativos para fluxos de cliente e advogado
